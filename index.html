¿Qué es Git?

Conoce la diferencia entre estas herramientas.

¿Qué es Git?

Git es un sistema de control de versiones que funciona como una "máquina del tiempo" para tu código. Te permite registrar cada cambio, volver a versiones anteriores y trabajar en equipo de forma organizada sin borrar el trabajo de otros. Al ser distribuido, cada programador tiene una copia completa del historial en su computadora.

¿Qué es GitHub?

GitHub es una plataforma en la nube que funciona como una red social para desarrolladores, permitiéndoles alojar sus proyectos de Git para que sean accesibles desde cualquier lugar. A diferencia de Git, que es la herramienta técnica para gestionar cambios, GitHub ofrece una interfaz visual para colaborar en equipo, revisar código ajeno, reportar errores y documentar software. Es el lugar donde la comunidad comparte código abierto y donde muchos programadores construyen su portafolio profesional para mostrar lo que saben hacer al mundo.

Sobre mis trabajos

Durante el desarrollo del curso, se realizarán diferentes actividades enfocadas en el diseño, desarrollo e implementación de proyectos, utilizando Java como lenguaje base y GitHub para la gestión profesional del código. Este portafolio organiza mis avances y permite acceder de forma sencilla a cada una de las tareas realizadas, reflejando mi progreso técnico y el uso de herramientas de control de versiones en cada entrega.
